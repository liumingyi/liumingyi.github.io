第一周：2个热身题目

1、只出现一次的数

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

Given a non-empty array of integers, every element appears twice except for one. Find that single one.

例如：输入：[2,2,1] 输出：1

我的笨方法实现：

思路 -> 从0开始拿出一个数（称为 目标数）和后面的数做对比，如果找到相同的数，则将其位置和目标数后一个位置交换，（即将相同的数放到一起）。如果对比完所有数也没有找到相同的，则目标数就是要找的single number；如此反复对比下去相同的数终将都排到一起，而数组中最后一个数字就是要找的single number。

2、求众数

给定一个大小为 n 的数组，找到其中的众数。众数是指在数组中出现次数大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在众数。

Given an array of size n, find the majority element. The majority element is the element that appears more than ⌊ n/2 ⌋ times.

You may assume that the array is non-empty and the majority element always exist in the array.

我的笨方法：

思路 -> 取出一个数（称为 目标数），与剩下的对比，将相同的挪到一起，一旦发现个数 > [ n/2 ]，即为众数（众数只可能有一个）。
